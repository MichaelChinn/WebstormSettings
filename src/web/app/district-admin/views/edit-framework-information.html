Select Frameworks Information

Select the evaluation framework used for teacher and principal evaluations and configure when the framework is made available to evaluators. In addition, when your district is using an Instructional Framework, you may optionally configure which framework views are visible throughout the application.

The choices are as follows:
  1. State and Instructional - State Default
  2. State and Instructional - Instructional Default
  3. State Only
  4. Instructional Only

#1 and #2 are used when the frameworks are presented on tabs, such as on the observation scoring screen. Both #1 and #2 would show both the State Rubrics and Instructional Rubrics tabs, but #1 would start with the State Rubrics tab active, whereas #2 would start with the Instructional Rubrics tab active.

#3 and #4 only show the one framework view. This would apply to screens where rubric alignment occurs, such as when configuring a user prompt, aligning an artifact to rubrics, or configuring the focus for an observation.
